# Тестовое задание для Datasub

## Используемые технологии
- React
- Axios
- React hook form
- Yup
- Node
- Express
- Mongoose
- Cors

## Чтобы запустить проект:
1. Склонировать репозиторий
2. Запустить backend
```
cd back
npm install
npm start
```
3. Запустить frontend
```
cd front
npm install
npm run build && npm run start
```
4. Открыть в браузере http://localhost:3000
